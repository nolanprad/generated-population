Plots de validation

Plots. Déposez vos exports dans un dossier img/ puis renseignez le src de chaque <img>. Tant qu'une image manque, un cadre gris affiche le nom de fichier attendu.

Tableaux de données

CSV. Deux façons d'alimenter les tableaux : (1) déposez vos fichiers dans un dossier data/ et listez-les dans app.js (constante CSV_FILES) — chargés automatiquement une fois le site en ligne ; ou (2) cliquez sur Charger un CSV ci-dessous pour l'afficher immédiatement (marche aussi en local). Le séparateur (, ; ou tabulation) est détecté automatiquement.

Les colonnes s'affichent dans l'ordre du fichier. Vous réagencerez ensuite.
